# 安装前准备

## 准备工具

在开始之前，您需要准备好一个空U盘（建议8GB以上），并且关闭电脑的安全启动功能。

## 调配分区

我们建议您通过专业的分区工具（如[DiskGenius]）进行分区。您需要用分区工具在电脑硬盘上腾出40GB（建议值）的**空闲**空间。我们不建议直接创建分区，而是在安装时由安装程序进行创建。

**请不要将Linux安装到任何NTFS文件系统的分区！Linux对NTFS的支持尚不完善！！！**

**任何分区操作都存在风险，请您在操作之前确保已经备份了重要数据！！！**
