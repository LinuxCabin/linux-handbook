# CachyOS

![Logo](https://cachyos.org/_astro/logo.DuMERIP6.svg)

## 简介

CachyOS 是一个基于 Arch Linux 优化的高性能滚动更新发行版。它最大的亮点是默认针对现代 CPU 架构（如 x86-64-v3, x86-64-v4）进行了专属编译优化，并默认搭载高度调优的内核（如 BORE 调度器）。CachyOS 不仅提供了极佳的桌面响应速度和游戏性能，还自带易用的图形化安装器与特制的系统调优工具，是追求极致硬件性能用户的首选。
