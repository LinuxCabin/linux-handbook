# 概念解释

欢迎来到“发行版”一节！我们在此整理了一些发行版的基本信息与使用时的常见问题。

以下是一些可能需要阐释的概念。

## 自动换源

简单来说，换源后下载速度：自动=手动=半自动=302，操作难度：自动=302（不换）<半自动<手动=302（换）

### 302

302指统一解析后重定向至较快软件源。不过，由于统一解析的服务器可能位于海外，在解析速度上可能略逊，但下载速度基本不受影响。

如：Fedora的源配置中，默认仓库为`https://mirrors.fedoraproject.org/`，该网址会自动将请求重定向到最快的镜像服务器。

### 半自动

半自动指需手动测速/切换但无需直接编辑配置文件。

如：Ubuntu不需要（但可以）通过更改源配置文件来换源，只需通过“软件和更新”即可选择镜像源。
