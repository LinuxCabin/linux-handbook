# Deepin/统信UOS

![Deepin](https://storage.deepin.org.cn/thread/202305231040315865_1.png)

## 简介

Deepin（深度操作系统）是一款源自中国、享誉全球的Linux发行版，以其优雅的设计和“开箱即用”的体验而闻名，是中国Linux小白用户的不二之选。然而，不少社区用户批评其缺乏创新，自定义性低，且Deepin的母公司曾有过“西装公司”丑闻。

统信UOS是Deepin的商用版本。

## 详细信息

 - 派系：Debian系
 - 发行者：武汉深之度科技有限公司/统信软件技术有限公司
 - 官网：https://www.deepin.org / https://www.chinauos.com/
 - 支持的系统架构：x86_64/arm64/Loong64/RISC-V等
