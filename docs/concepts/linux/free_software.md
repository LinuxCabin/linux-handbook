# 自由软件

英文：Free Software

## 概述

简单来说，自由软件指源码开放，用户可以按照许可证条款对软件及其源代码进行随意的研究、修改、分发、二次分发等行为。

以下分别出示GNU自由软件理念和BSD自由软件理念

（GNU）
> 0：无论用户出于何种目的，必须可以按照用户意愿，自由地运行该软件。

> 1：用户可以自由地学习并修改该软件，以此来帮助用户完成用户自己的计算。作为前提，用户必须可以访问到该软件的源代码。 

> 2：用户可以自由地分发该软件的拷贝，这样就可以助人。 

> 3：用户可以自由地分发该软件修改后的拷贝。借此，用户可以把改进后的软件分享给整个社区令他人也从中受益。作为前提，用户必须可以访问到该软件的源代码。 

（BSD）
> 0：无论用户出于何种目的，必须可以按照用户意愿，完全自由地运行该软件，且没有任何使用场景的限制。   

> 1：用户可以自由地学习并修改该软件，以满足用户自己的计算需求。作为前提，用户必须可以公开访问到该软件的源代码，且原作者不对修改行为作任何审计或干涉。   

> 2：用户可以自由地分发该软件的原始拷贝或修改后的拷贝，用户可以自由选择以何种许可证分发，只需保留原作者的版权声明即可。 

> 3：用户可以自由地分发该软件修改后的版本，且不承担任何向社区回馈代码的义务。   

具体的定义，我们建议您参考[GNU官网](https://www.gnu.org/philosophy/free-sw.zh-cn.html)和[BSD许可证](https://opensource.org/license/bsd-2-clause)
