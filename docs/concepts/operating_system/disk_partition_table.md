# 磁盘分区表

英文：Disk Partition Table

## 概述

磁盘分区表是记录硬盘分区信息的数据结构。目前主流的分区表格式有两种：传统的老式 MBR 和现代通用的 GPT。选择哪种分区表直接决定了您能使用的最大硬盘容量以及支持的分区数量。

- **MBR**：传统方案，仅支持4个主分区（或3主+1扩展），最大2TB容量，易受限制，适用于旧系统（Windows 7 前使用）。
- **GPT**：现代标准，支持128个分区，容量可达18EB，兼容UEFI引导，数据更安全（冗余分区表），适合大容量硬盘和新系统。

???+ note "如何选择？"
    目前在UEFI工况下，闭眼选GPT即可，如果你的机器不支持UEFI，请选择MBR分区表
