# 分区

英文：Partition

## 概述

硬盘分区是使用分割编辑器（比如DiskGenius，Windows的磁盘管理等等等等）在磁盘上划分几个逻辑部分，也就是一块盘上的多个分区，越多分区，也就有更多不同存储数据的地方，通过分区，可以将文件的性质区分得更细，但是对空间管理、访问许可与目录搜索的方式，依属于安装在分区上的文件系统来控制。

简单来说，分区就是在单块/多块磁盘上划分出不同的数据存储区域，这些区域可以有不同的文件系统，相互独立。

???+note "关于分区连续性"
    一般情况下，分区都是**连续的**，因此调整分区时需注意分区的的顺序位置，来回调整分区极易造成**数据损坏！！！**
