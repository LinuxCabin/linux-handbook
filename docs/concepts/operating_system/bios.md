英文：Basic Input/Output System（BIOS）

## 概述

BIOS即基本输入输出系统，亦称为ROM BIOS、System BIOS、PC BIOS，是在通电启动阶段执行硬件初始化，以及为操作系统提供运行时服务的固件。BIOS预安装在个人电脑的主板上，是个人电脑启动时加载的第一个软件。现在，BIOS的作用是初始化和测试硬件组件，以及从大容量存储设备（如硬盘）加载启动程序（如GRUB，bootmgrfw等），并由启动程序加载操作系统；当加载操作系统后，BIOS通过系统管理模式为操作系统提供硬件抽象（可以理解为CPU的不同角色，分实模式，保护模式，长模式）。

# UEFI

英文：Unified Extensible Firmware Interface（UEFI）

## 概述

UEFI即统一可扩展固件接口，是基于BIOS的衍生替代方案，其作用主要来定义操作系统与系统固件之间的软件界面，作为BIOS的替代方案。可扩展固件接口负责加电自检（POST）、联系操作系统以及提供连接操作系统与硬件的固件层产品。

目前，统一可扩展固件接口（UEFI）正在逐步取代旧式BIOS，但仍有很多产品支持兼容模式，通过Legacy选项，但类似这种的支持逐渐减少。目前，UEFI与兼容支持模式（CSM）成为主流，但最新的产品已经不再支持兼容模式，仅支持UEFI。 

???+ note "如何选择？"
    支持且无严重Bug的情况下选UEFI，更加现代，安全，快速。如果不支持或存在严重bug的情况下选择BIOS。
